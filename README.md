# Menu

Digital menu for https://menu.f2p.be

Built with Jekyll + GitHub Pages. Menu data lives in `_data/menu.yml`.

## Icons

This site uses **Font Awesome Free** icons. Browse available free icons at:

https://fontawesome.com/search?ic=free-collection

Icon classes are set in `_data/menu.yml` (per section and per item). Use the `fa-solid fa-` prefix.
